Internet weit verbreitete Auszeichnungssprache (*Markup-Language*).
Andere bekannte Ausszeichnungssprachen sind:

- Hypertext Markup Language (HTML)
- Extensible Markup Language (XML)
- yet another Markup Language ( YAML; YML)
